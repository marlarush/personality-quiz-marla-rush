# Personality Quiz (Java Swing) — Introvert vs Extrovert

**Solo project by Marla Rush**

A Java Swing app where users answer multiple-choice questions. Each answer awards points to personality traits (Introvert/Extrovert). The final result is displayed with a brief description and a restart option.
